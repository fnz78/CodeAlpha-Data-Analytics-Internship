# CodeAlpha Internship - Task 3 (Data Visualization)

## Objective
Transform scraped book data into meaningful visual insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations Created
- Price Distribution
- Rating Distribution
- Price vs Rating Boxplot
- Average Price by Rating
- Top 5 Expensive Books
- Correlation Heatmap

## Key Insights
- Most books priced £40–60
- Rating 3 most common
- Weak correlation between price and rating
